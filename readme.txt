== Description ==

Bulk Remove Users helps to hold clean your list of users.

Add in the plugin's textarea a list of email of users registered commq separated, and bulk delete these users in one-click.


== Installation ==


1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Bulk Remove Users to use the plugin


== Frequently Asked Questions ==

= How many users I can remove simultaneously =

You can bulk remove all the users you can do. 
N.B: With a very long list response times may increase significantly.

= What are the user roles that I can remove? =

AYou can only remove the subscribers.
